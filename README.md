# FruitAI 🍎

AI-Based Fruit Freshness and Expiry Prediction Using Images

## Features
- Fruit Freshness Detection
- Expiry Days Prediction
- CNN-Based Classification
- Flask Web Application

## Technologies
- Python
- TensorFlow
- Keras
- OpenCV
- Flask
