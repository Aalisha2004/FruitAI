# 🍎 FruitAI – AI-Based Fruit Freshness and Expiry Prediction Using Images

A deep learning-based web application that detects fruit freshness and estimates remaining shelf life using image analysis. The system uses CNN, TensorFlow, OpenCV, and Flask to classify fruits as fresh or rotten and predict expiry days.

## 📌 Project Summary

* **Project Type:** Final Year Mega Project
* **Model:** Convolutional Neural Network (CNN)
* **Technologies:** TensorFlow, Keras, OpenCV, Flask
* **Accuracy:** 95.96%
* **Goal:** Fruit Freshness Detection & Expiry Prediction

## 🧠 Classes Detected

* Fresh Apple / Rotten Apple
* Fresh Banana / Rotten Banana
* Fresh Orange / Rotten Orange
* Fresh Mango / Rotten Mango
* Fresh Grapes / Rotten Grapes

**Total Classes: 10**

## 🚀 Features

* Fresh vs Rotten Classification
* Freshness Percentage
* AI Confidence Score
* Quality Grading (A+, A, B, Reject)
* Shelf Life & Expiry Prediction
* Multiple Image Upload
* Interactive Dashboard
